# Program Daftar Nilai Mahasiswa

Program ini dibuat untuk memenuhi *Tugas Praktikum* dengan ketentuan sebagai berikut:

- Program dibuat menggunakan **Dictionary**
- Terdapat menu pilihan:
  1. Tambah Data  
  2. Ubah Data  
  3. Hapus Data  
  4. Tampilkan Data  
  5. Cari Data  
- Nilai akhir dihitung dari 3 komponen:
  - Tugas: 30%
  - UTS: 35%
  - UAS: 35%
- Program menampilkan data mahasiswa beserta nilai akhirnya
- Terdapat flowchart dan penjelasan program
